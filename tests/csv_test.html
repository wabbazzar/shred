📋 CSV Format Specification
Reference for the expected CSV format and column requirements.
Required Column Headers:
program_name,week,day,day_name,section,exercise_name,prescribed_sets,actual_sets,prescribed_reps,actual_reps,prescribed_weight,actual_weight,prescribed_time,actual_time,exercise_type,notes
Example Row:

Format Requirements
✅ All 16 required columns must be present
✅ Week numbers must be 1-6
✅ Day numbers must be 1-7
✅ Exercise types: strength, cardio, bodyweight, timed
✅ Numeric fields must be valid numbers or empty
